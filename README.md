# Buddiz

Buddiz is a socialisation progressive web app, meant to help people meet dans spend time together around activities. The main features on this project are : 
- the interactive map, referencing the activities
- an activities list in the form of cards
- a single activity vue
- an activity creation form
- a personality test form
- a profile management page

## Run the project

To run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

To see the web site online, go to https://buddiz.vercel.app
